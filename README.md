# Adminka (Work in progress)

> GraphQL server with dashboard boilerplate

## 🚀 Backend Setup

Firt, install all dependencies

```bash
yarn
```

**Don't forget** to fill _.env_ config in root using _.env.example_ file as an example

And run:

```bash
yarn start
```
